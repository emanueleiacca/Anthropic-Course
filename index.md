# Anthropic Academy

A topic-first knowledge base for studying Claude, agent systems, MCP, prompting, API workflows, and Claude Code.

## Start Here

1. Begin with [Topics](Topics/index.md) for the main concept map.
2. Use [Foundations](Topics/Foundations/index.md) and [Prompting](Topics/Prompting/index.md) to build the core mental model.
3. Move into [API & Tools](Topics/API-Tools/index.md) and [Agents & MCP](Topics/Agents-MCP/index.md) for implementation patterns.
4. Finish with [Claude Code](Topics/Claude%20Code/index.md) and [Ethics & Safety](Topics/Ethics-Safety/index.md) for product workflows and safeguards.
